# FlowBoard

AI-assisted collaborative visual board for teams to organize ideas, research, data, and workflows as connected flowchart nodes.

## Setup

```bash
# Install dependencies (pnpm recommended; npm may have issues with vite in some environments)
pnpm install
# or
npm install
```

## Build

```bash
pnpm run build
# or
npm run build
```

## Features

- **Landing Page** — Hero, features, pricing, testimonials, footer
- **Auth** — Login, signup, password reset, email verification
- **Dashboard** — Collapsible sidebar, overview, projects, quick actions
- **Board** — Visual canvas with toolbar, node palette, AI agent panel
- **Templates** — Catalog with filters and apply
- **Import/Export** — CSV/JSON import, PNG/PDF/JSON/CSV export
- **Settings** — Profile, organization, AI, integrations, notifications
- **Admin** — User management, billing, audit logs, feature flags
- **Legal** — Privacy policy, terms of service
- **Error pages** — 404, 500

## Tech Stack

- React 19, React Router 6, TypeScript
- Tailwind CSS v3, Shadcn/Radix UI
- TanStack React Query, React Hook Form, Zod
- Sonner, Recharts, Lucide React

## Design

Follows FlowBoard UI Guide: primary #0B63FF, Inter typography, 6px radius, card shadows.
